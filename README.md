# LOLBot
## _Your League of Legends assistant_

LOLBot is an AI powered chatbot to help improve your League of Legends experience

## Installation 

It should just work


## Usage

- Input your League of Legends summoner name in the "Summoner Name" field 
-- Use **Jessicaclan** if you dont have one
- Press "Find summoner" and wait for the "Ready for message" checkmark
- You can now ask question regarding your latest game
--Examples: Who did i do?, which items did i build?
